4. Write a code to calculate area of a rectangle using inner function. In this case outer function should accept parameter length and inner function should accept parameter breadth.

Answer:
        function outfun(l){
            function innerfun(b){
                return l*b;
            }
            return innerfun;
        }
        const l=5;
        const b=10;
        const area = outfun(l)(b);
        console.log(area);
